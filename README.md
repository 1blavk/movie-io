# Movie I/O (Input/Output)

- REST API bilan ishlash: fetch yoki axios yordamida TMDB ma'lumotlarini olish va asinxron funksiyalar bilan ishlash.
- React Hooks: useState (ma'lumotlarni saqlash) va useEffect (sahifa yuklanganda API'ga so'rov yuborish) tushunchalari.
- Dinamik Routing: Har bir film uchun alohida sahifa ochish (react-router-dom yordamida movie/:id strukturasi).
- UI/UX va Komponentlar: Filmlar ro'yxati, qidiruv paneli va film kartochkalarini alohida komponentlarga ajratish.
- Deployment: Tayyor loyihani Vercel platformasiga joylashtirish.
